Juego del Ahorcado en Python
Este proyecto es una implementación del clásico Juego del Ahorcado en Python, con diferentes modos de juego y estadísticas.

Modos de juego
  Un jugador
La computadora selecciona una palabra aleatoria y el jugador debe adivinarla antes de quedarse sin intentos.

  Contra la computadora
El jugador escribe una palabra secreta y la computadora intentará adivinarla de manera aleatoria.

  Multijugador (2 jugadores)
El primer jugador escribe la palabra y el segundo jugador debe adivinarla antes de que se agoten los intentos.

  Estadísticas
El juego muestra el resultado de la última partida jugada (modo, palabra y quién ganó).

  Requisitos
Python 3 instalado en tu computadora.
No requiere librerías externas, solo se usa el módulo estándar random.

  Cómo ejecutar el juego
Guarda el archivo en tu computadora con el nombre ahorcado.py.
Abre una terminal o consola en la carpeta donde guardaste el archivo.
Ejecuta el juego con el comando:
 "python ahorcado.py"
Sigue las instrucciones en pantalla y selecciona el modo de juego que prefieras.

  Notas
Cada partida tiene un número fijo de 5 intentos.
El juego controla letras repetidas para no descontar intentos de más.
El menú permite jugar varias veces seguidas hasta que decidas salir.
